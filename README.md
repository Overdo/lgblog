#### this is a flask blog system

#### require

- Python: 3.5
- flask：0.12.2
- Mysql：57

demo： http://lgtal.top

#### usage

```
git clone https://github.com/Overdo/lgblog.git
cd lgblog
pip install -r requirements.txt

# create fake data
python manager.py init_database

# runserver
python runserver.py

# then 
INFO:werkzeug: * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
```
